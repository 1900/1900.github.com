使用python监控进程

2016年02月04日 标签：Python

eg.
监控nginx

# !/usr/bin/env python
import os, sys, time

while True:  
    time.sleep(3)
    try:
        ret = os.popen('ps -C nginx -o pid,cmd').readlines()
        if len(ret) < 2:
            print "nginx process killed, restarting service in 3 seconds."
            time.sleep(3)
            os.system("service nginx restart")
    except:
        print "Error", sys.exc_info()[1]
